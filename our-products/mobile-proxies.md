# Мобільні проксі

<mark style="color:purple;">Мобільні проксі</mark> розміщуються на роутерах із реальними SIM-картами. Тип і якість з'єднання ідентичні звичайному мобільному інтернету через стільникового оператора — точно такому ж, як на вашому смартфоні. Підтримують <mark style="color:purple;">UDP</mark>, пропонують багатий вибір локацій та операторів.

{% hint style="info" %}
**Мобільні проксі працюють на будь-якому пристрої.** Назва «мобільні» відображає лише спосіб підключення через SIM-карту, а не тип кінцевого пристрою. Їх однаково добре використовувати на ПК, ноутбуці, в антидетект-браузері.
{% endhint %}

{% hint style="warning" %}
**Специфічний продукт — для тих, хто розуміє навіщо.** Трафік іде через реальну SIM-карту, тому швидкість може змінюватися — це норма, а не несправність. Якщо не впевнені, чи підійдуть мобільні проксі під ваше завдання, уточніть у [live-чаті](../contact-us.md) або розгляньте [ISP проксі](isp-proxies.md) (стабільна швидкість, домашні IP) чи [Резидентські проксі](residential-proxies/README.md) (великий пул, безліч сесій).
{% endhint %}

{% hint style="danger" %}
1\) Зміна відбитка пристрою (p0f) доступна не на всіх операторах. Повний список обмежень дивіться на сторінці [Обмеження](restrictions.md).

2\) Недоступно користувачам РФ без використання VPN
{% endhint %}

## Опис полів замовлення

Розглянемо поля <mark style="color:purple;">замовлення</mark>:

<figure><img src="../.gitbook/assets/image (20) (1).png" alt=""><figcaption></figcaption></figure>

<mark style="color:purple;">Proxy info</mark> - Найменування продукту

<mark style="color:purple;">Reset URL</mark> - Посилання для зміни IP адреси на підключенні

<mark style="color:purple;">Login</mark> - Логін проксі

<mark style="color:purple;">Password</mark> - Пароль проксі

<mark style="color:purple;">Order status</mark> - статус замовлення, може мати статуси:

* <mark style="color:green;">**Active**</mark> - Активне замовлення
* <mark style="color:orange;">**On-Hold**</mark> - Очікування оплати замовлення при закінченні терміну оренди
* <mark style="color:red;">**Cancelled**</mark> - Скасоване замовлення

<mark style="color:purple;">Proxy status</mark> - статус замовлення, може мати статуси:

* <mark style="color:green;">**Active**</mark> - Активне замовлення
* <mark style="color:$danger;">Disconnected</mark> - Вимкнений, не активний порт

{% hint style="danger" %}
**Після купівлі або відсутності активності на порту, потрібно його **<mark style="color:$success;">**активувати**</mark>**, якщо статус проксі - **<mark style="color:$danger;">**Dosconnected**</mark>**, то поки ви їх не активуєте, вони працювати не будуть!**

**Активувати проксі можна через** <mark style="color:purple;">**Reset URL**</mark> **або кнопку** <mark style="color:purple;">**Restart Proxy**</mark>**.**
{% endhint %}

<mark style="color:purple;">Next Due Date</mark> - Наступна дата списання

<mark style="color:purple;">Copy proxy</mark> - кнопка для копіювання проксі в буфер обміну

<mark style="color:purple;">Re-generate</mark> - Зміна пароля на проксі\
\
<mark style="color:purple;">Restart</mark> - Запуск або зміна IP, аналогічна <mark style="color:purple;">Reset URL</mark> адресою

## Для яких завдань підходить

Соціальні мережі та мультиакаунтинг, mobile ad verification, перевірка мобільної реклами, більшість криптобірж, Polymarket, тестування мобільних сайтів та застосунків, web scraping мобільних версій сайтів, SEO моніторинг мобільної видачі, e-commerce моніторинг мобільних цін, геотаргетоване тестування контенту, travel scraping мобільних тарифів, brand monitoring у мобільному середовищі.

## Плюси та мінуси Мобільних проксі

#### <mark style="color:green;">Плюси:</mark>

* **Вибір конкретного оператора** — підключення від потрібного стільникового провайдера
* **Зміна IP за Reset URL** — ротація не частіше одного разу на хвилину
* **Підтримка p0f** — доступна на більшості локацій (див. [Обмеження](restrictions.md))
* **Підтримка UDP**
* **Гнучкий термін оренди** — від одного дня до місяця
* **Порт в одні руки** — одна SIM-карта, один користувач

#### <mark style="color:red;">Мінуси:</mark>

* **Можливі просідання швидкості** при перевантаженні стільникової станції оператора — рідко, але можливо
* **Складний продукт для новачків** — рекомендуємо уточнити в [підтримці](../contact-us.md) перед купівлею
* **Підміна p0f на macOS / iOS** знижує швидкість каналу через складність алгоритму
* **Динамічний IP** — адреса може змінитися з ініціативи оператора будь-якої миті
* **Одна сесія одночасно** — один порт тримає один IP. Якщо потрібно підключити кілька пристроїв _одночасно_ (не по черзі, а саме в один момент) — придбайте окремий порт або розгляньте [Резидентські проксі](residential-proxies/README.md)
