# Unlimited Residential proxy

Безлімітні резидентські проксі. За функціональністю повністю ідентичні звичайним резидентським проксі, але без обмежень по трафіку.

Підходять для сценаріїв із великим споживанням трафіку, де використання звичайних резидентських проксі з оплатою за гігабайт стає невигідним.

{% hint style="info" %}
**Обмеження:** \
Не більше 5000 активних з'єднань одночасно. Ліміт можна збільшити на запит.\
Максимальна швидкість на замовлення = 75 **MB/s, на стандартному плані**
{% endhint %}
