# Як перевірити працездатність проксі

Якщо проксі не підключаються або ви хочете переконатися, що все працює, є два прості способи перевірки.

***

## Спосіб 1: Proxy Tester на сайті

Найшвидший варіант.

Відкрийте наш [Proxy Tester](https://proxyshard.com/proxy-tester), вставте проксі у поле та натисніть **Test Proxy**.

Докладніше про можливості та формати читайте у статті [Proxy Tester](../../our-products/proxy-tester.md).

***

## Спосіб 2: Перевірка через curl (командний рядок)

Цей спосіб підходить, якщо потрібно швидко перевірити підключення прямо з комп'ютера. Працює на Windows, macOS та Linux.

### Windows

Відкрийте командний рядок: натисніть <mark style="color:blue;">WIN+R</mark>, введіть `cmd` та натисніть Enter.

### macOS / Linux

Відкрийте Terminal (на macOS через Spotlight або папку Applications > Utilities).

***

### Перевірка HTTP проксі

```bash
curl -x http://ЛОГИН:ПАРОЛЬ@IP_ПРОКСИ:ПОРТ -I https://www.google.com
```

Приклад із реальними даними:

```bash
curl -x http://KWFFEKf:LDLFEPF@88.99.123.233:44454 -I https://www.google.com
```

### Перевірка SOCKS5 проксі

```bash
curl --socks5 ЛОГІН:ПАРОЛЬ@IP_ПРОКСІ:ПОРТ -I https://www.google.com
```

Приклад:

```bash
curl --socks5 KWFFEKf:LDLFEPF@88.99.123.233:44454 -I https://www.google.com
```

***

### Як зрозуміти результат

{% hint style="success" %}
**Проксі працює.** Якщо у відповіді ви бачите `HTTP/1.1 200 OK` або `HTTP/2 200`, значить з'єднання пройшло успішно.
{% endhint %}

{% hint style="danger" %}
**Проксі не працює.** Якщо ви отримали помилку на кшталт `Connection refused`, `Connection timed out` або `Could not resolve proxy`, значить підключитися не вдалося.
{% endhint %}

### Що робити, якщо не підключається

1. Перевірте логін, пароль, IP та порт. Часта помилка: плутають **HTTP** та **SOCKS5** порти
2. Спробуйте підключитися через VPN, щоб унеможливити блокування з боку вашого провайдера
3. Зачекайте 2-3 хвилини після покупки: серверам потрібен час на оновлення
4. Якщо нічого не допомогло, зверніться до [підтримки ](../../contact-us.md)
