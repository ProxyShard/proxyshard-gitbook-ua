## Встановлення ProxyCap**.**

Перейдіть по [посиланню](https://www.proxycap.com/download.html) для завантаження ProxyCap

Виконайте установку та запустіть програму.<br>

Далі перейдіть у вкладку <mark style="color:purple;">Proxies</mark> і додайте нове підключення через значок ![](<../../.gitbook/assets/image (98).png>)

## **Налаштування профілю підключення**

Вкажіть проксі із замовлення у відповідні поля

{% hint style="info" %}

## **Налаштування правил маршрутизації**

Для роботи проксі потрібно обов'язкове додавання правила маршрутизації.

Щоб додати правила, відкрийте вкладку "<mark style="color:purple;">Rules</mark>" та вкажіть налаштування як на скріншоті, обов'язково встановивши <mark style="color:purple;">Rule Action</mark>, <mark style="color:purple;">Proxy</mark> та інші параметри за потребою.

Ви можете встановити окреме проксування процесів, задавши шлях програми в полі <mark style="color:purple;">Program</mark>
