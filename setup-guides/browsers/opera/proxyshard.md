# ProxyShard Extension

Наше фірмове розширення <mark style="color:purple;">**ProxyShard – Smart Proxy Manager**</mark> працює в Opera, але для встановлення розширень з Chrome Web Store потрібен додатковий крок.

## Встановлення з Chrome Web Store в Opera

1. Відкрийте Opera Add-ons та встановіть розширення [**Install Chrome Extensions**](https://addons.opera.com/extensions/details/install-chrome-extensions/).
2. Після встановлення відкрийте Chrome Web Store та встановіть ProxyShard як зазвичай.
