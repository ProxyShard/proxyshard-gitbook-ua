# Safari

На жаль, робочого розширення для браузера <mark style="color:purple;">Safari</mark> немає, він не підтримує роботу з доповненнями такого функціоналу, але ви можете розглянути альтернативні варіанти:\
1\) [Налаштування для macOS](../macos/)\
2\) [Антидетект браузери](../antidetect-browsers/)\
3\) Встановлення браузера [Chrome](chrome/), [Mozilla Firefox](mozilla-firefox/) або інших браузерів з підтримкою розширень (Opera, Brave, Vivaldi...)
