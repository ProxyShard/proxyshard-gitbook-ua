# ProxyShard Extension

Наше фірмове розширення <mark style="color:purple;">**ProxyShard – Smart Proxy Manager**</mark> дозволяє керувати проксі прямо з Mozilla Firefox. Встановлюється напряму з **Firefox Add-ons**.

{% embed url="https://addons.mozilla.org/en-US/firefox/addon/proxyshard-smart-proxy-manager/" %}

{% hint style="success" %}
У версії для Mozilla Firefox підтримується робота з <mark style="color:purple;">**SOCKS5**</mark> через розширення.
{% endhint %}
