# ProxyShard Extension

Наше фірмове розширення <mark style="color:purple;">**ProxyShard – Smart Proxy Manager**</mark> працює в Microsoft Edge без обмежень, він побудований на рушії Chromium та підтримує розширення з **Chrome Web Store**.

{% hint style="info" %}
Під час першого встановлення розширення з Chrome Web Store Edge може запропонувати «**Дозволити розширення з інших магазинів**», підтвердьте цей дозвіл.
{% endhint %}
