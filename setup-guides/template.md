3. **Додавання списку проксі із замовлення до програми.**

Для додавання проксі в ???, вам необхідно відкрити ваше замовлення на сайті та скопіювати проксі:

{% hint style="info" %}
Зауважте, що тестове підключення здійснюється через SOCKS5. Щоб дізнатися порт підключення SOCKS5, потрібно:\
<mark style="color:purple;">Datacenter\ISP</mark> - встановити перемикач на ![](<../.gitbook/assets/image (38).png>)в полі <mark style="color:purple;">Proxy list</mark>



<mark style="color:purple;">Residential proxy</mark> - У налаштуваннях замовлення вказати "<mark style="color:purple;">Protocol</mark>" = SOCKS5



<mark style="color:purple;">Mobile proxy</mark> - Не вимагають зміни порту або інших налаштувань, підключення по одному порту працює як по HTTP так і по SOCKS5
{% endhint %}

Приклад для <mark style="color:purple;">Датацентр/ISP</mark> проксі:\
Відкрийте придбаний замовлення та скопіюйте підключення з <mark style="color:purple;">Proxy List</mark>.

<figure><img src="../.gitbook/assets/image (35).png" alt=""><figcaption></figcaption></figure>

Приклад для <mark style="color:purple;">Резидентських проксі</mark>:

Вкажіть потрібні параметри проксі, наприклад країну/протокол, і натисніть "<mark style="color:purple;">Generate proxy</mark>"

<figure><img src="../.gitbook/assets/image (260).png" alt=""><figcaption></figcaption></figure>

Скопіюйте отримані проксі з <mark style="color:purple;">Proxy List</mark> або скопіюйте окремі поля підключення, якщо програма не підтримує вставку рядком.

Нижче наведено скріншот, яке з полів відповідає за "Server":"Port":"Login":"Password"
